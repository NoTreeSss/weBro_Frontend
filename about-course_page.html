Стоимость: бесплатно
Сложность: низкая
Время прохождения: ~10 мин
Кому подойдет этот курс
  • Тем, кто хочет узнать, как создаются веб-сайты

  • Создавать проект и подготавливать свое рабочее пространство в Visual Studio Code

  • Стилизовать свой сайт и адаптировать его под разные устройства с помощью CSS
